### Sobre o projeto

Fiz uma aplicação de webchat usando Javascript Vanilla no front-end e Node.js, Express e Socket.io para fazer a comunicação com o backend.
Usei mongodb para fazer a persitencia dos dados dados.

- Fazer deploy
